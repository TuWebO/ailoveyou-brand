# Typography

## Purpose

Typography is one of the primary ways AI Love You communicates its personality.

It should feel:

- Warm
- Calm
- Modern
- Approachable
- Confident
- Clear

It should never feel:

- Cold
- Corporate
- Futuristic
- Playful
- Aggressive

The typography should disappear behind the message. Reading should feel effortless.

---

# Primary Typeface

## Geist

Geist is the primary typeface for AI Love You.

Reasons for choosing Geist:

- Open source
- Highly legible
- Excellent at small sizes
- Designed for digital interfaces
- Modern without feeling trendy
- Wide language support
- Available as a variable font

The entire interface should be built using Geist.

Avoid mixing multiple typefaces unless there is a strong design reason.

---

# Font Scale

The following scale should be used whenever possible.

| Usage | Size |
|--------|-----:|
| Display | 64 px |
| H1 | 48 px |
| H2 | 36 px |
| H3 | 30 px |
| H4 | 24 px |
| Body Large | 20 px |
| Body | 16 px |
| Small | 14 px |
| Caption | 12 px |

---

# Font Weights

Prefer a small number of weights.

| Weight | Usage |
|---------|------|
| 400 | Body text |
| 500 | UI labels |
| 600 | Headings |
| 700 | Rare emphasis |

Avoid using many different weights on the same page.

---

# Line Height

Reading comfort is more important than compact layouts.

Recommended values:

- Headings: 1.2
- Body text: 1.5
- Long articles: 1.6

---

# Letter Spacing

Letter spacing should be subtle.

Avoid dramatic tracking.

Display text may use slightly tighter spacing.

Body text should use the default spacing provided by the typeface.

---

# Alignment

Default alignment is left.

Centered text should only be used:

- Hero sections
- Empty states
- Marketing headlines

Avoid justified text.

---

# Emphasis

Use hierarchy instead of decoration.

Prefer:

- Weight
- Size
- Whitespace

Avoid:

- Underlines
- All caps
- Excessive bold text
- Multiple colors

---

# Accessibility

Typography should always prioritize readability.

Guidelines:

- Minimum body size: 16 px
- Maintain sufficient contrast
- Avoid extremely light font weights
- Avoid long lines of text
- Use semantic heading levels

---

# Writing Style

Typography and voice work together.

Sentences should be:

- Short
- Clear
- Friendly
- Direct

Avoid unnecessary technical jargon.

The interface should feel like a thoughtful conversation.

---

# Guiding Principle

People should notice what they read, not the font they are reading.

If the typography calls attention to itself, it is probably too decorative.
